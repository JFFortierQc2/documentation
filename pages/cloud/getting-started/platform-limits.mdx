# Limits and Quotas of Botpress Cloud

At Botpress, we have a set of limits and quotas to ensure the stability of our platform. Here is a list of the current limits and quotas:

| **Type** |                          **Name** | **Amount** | **Description**                                                              |
| :------- | --------------------------------: | ---------- | ---------------------------------------------------------------------------- |
| Limit    |                 Bot Configuration | 20KB       | The maximum size for the bot settings                                        |
| Limit    |        Bot Installed Integrations | 30         | The maximum number of installed integrations per bot                         |
| Limit    |     Bot Integration Configuration | 20KB       | The maximum size for the integration settings                                |
| Limit    |              Bot Recurring Events | 5          | The maximum number of recurring events per bot                               |
| Limit    |  Bot Recurring Event Payload Size | 128KB      | The maximum size for the recurring event payload                             |
| Limit    |                  Event Definition | 20         | The maximum number of event definitions per bot                              |
| Limit    |              Integration Channels | 10         | The maximum number of integration channels                                   |
| Limit    | Integration Channel Message Types | 20         | The maximum number of message types per integration channel                  |
| Limit    |           Integration Identifiers | 50         | The maximum size for the integration identifier                              |
| Limit    |              Integration Versions | 50         | The maximum number of integration versions                                   |
| Limit    |     Issue Event Item Payload Size | 128KB      | The maximum size for the issue event item payload                            |
| Limit    |              Message Payload Size | 128KB      | The maximum size for the message payload                                     |
| Limit    |                       Schema Size | 20KB       | The maximum size for the integration schemas                                 |
| Limit    |                 State Definitions | 5          | The maximum number of state definitions per bot                              |
| Limit    |           Conversation State Size | 128KB      | The maximum size for the conversation state                                  |
| Limit    |                   Tag Definitions | 50         | The maximum number of tag definitions per bot                                |
| Quota    |                Invocation Timeout | 60s        | The maximum time a bot can run before timing out                             |
| Quota    |                      Storage Size | 100MB      | The maximum storage size per bot (media and Knowledge Base files)            |
| Quota    |                    Amount of Bots | 5          | The maximum amount of bots per workspace                                     |
| Quota    |                 Amount of Members | 3          | The maximum amount of members that can be added to a workspace               |
| Quota    |            Knowledge Base Vectors | 5000       | The maximum amount of vectors (content chunks) in a Knowledge Base per bot   |
| Quota    |                        Table Rows | 5000       | The maximum amount of rows in a table per bot                                |
| Quota    |                  Invocation Calls | 25000      | The maximum amount of times a bot can be invoked per month                   |
| Quota    |                   Cognitive Calls | 4000       | The maximum amount of times a bot can call cognitive (AI) services per month |
| Quota    |                    Bot Rate Limit | 100        | The maximum amount of times a bot can be invoked per minute                  |
| Quota    |                Owned Integrations | 20         | The maximum amount of custom integrations that can be created per workspace  |
| Quota    |                 LLM Model Credits | 5000       | The maximum amount of AI model credits that can be used per month            |
