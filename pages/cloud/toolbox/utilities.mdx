# Utility Cards 

## Comment

The Comment card is used to add a comment to the flow. It is useful to explain the flow to other people or to add notes to yourself.

Pick the Comment Card from the Utility Cards section in the Toolbox and drop it on the flow. Click on the card to edit the comment.

## Video

The Video card is used to add a video to the flow. The Video Card can be used to add a video from YouTube URL. 

## Image

The Image card is used to add an image to the flow. The Image Card can be used to add an image from your local machine.
